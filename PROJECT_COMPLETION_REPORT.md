## 📝 环境配置

### 必需配置
```env
TELEGRAM_BOT_TOKEN=your_bot_token
TELEGRAM_CHAT_ID=your_chat_id
SMTP_HOST=smtp.qq.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=your_email@qq.com
SMTP_PASS=your_authorization_code
SMTP_FROM=工单系统 <your_email@qq.com>
JWT_SECRET=<32+ 字符随机密钥>
```

### 可选配置
```env
JWT_EXPIRES_IN=24h
PORT=3000
HTTP_PROXY=
HTTPS_PROXY=
```
